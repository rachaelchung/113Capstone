Connects the extension to your backend (same account as the web app). The API URL is chosen automatically: local Flask first when it responds, otherwise the production URL set in extension/background.js (same as js/apiBaseConfig.js).

Google (and “continue in browser”) After you sign in on your site, open app.html (or the page that saves your session) in this same browser. The extension copies your session and API base from the page automatically. Refresh the app tab once if needed, then use “Verify session” below. Deployed site: your exact origin (e.g. https://your.app/*) must be listed in the extension manifest.json under content_scripts next to the localhost entries, then reload the extension.
